# Veromnuka

Это простая статическая версия сайта Veromnuka — личного бренда/проекта Вероники Житниковой про бренд-стратегию, дизайн упаковки FMCG, айдентику, презентации, каталоги, B2B-сайты и визуальные материалы для продаж.

Сайт работает без React, Next.js, CMS, базы данных и сборки: только HTML, CSS и JavaScript.

## Как открыть сайт

1. Скачайте или откройте папку проекта на компьютере.
2. Найдите файл `index.html`.
3. Откройте его двойным кликом в браузере.

## Из каких файлов состоит сайт

- `index.html` — структура страницы, SEO-теги, первый экран, блок «С какой задачей приходят», проекты, блок «Обо мне» и контакты.
- `styles.css` — внешний вид сайта: цвета, сетка, карточки, адаптивная мобильная версия.
- `script.js` — поиск, точные фильтры по категориям и вывод карточек на страницу.
- `opportunities.js` — список карточек направлений работы.
- `README.md` — эта инструкция.

## Где менять SEO-теги

SEO-основа находится в начале файла `index.html`, внутри тега `<head>`:

```html
<title>Veromnuka — бренд-стратегия, дизайн упаковки FMCG и визуальные системы для продаж</title>
<meta name="description" content="Veromnuka помогает брендам собрать стратегию, упаковку, айдентику, презентации, каталоги, B2B-сайты и визуальные материалы для запуска, продаж и роста заметности.">
```

Чтобы изменить заголовок вкладки браузера и сниппет для поисковых систем, редактируйте именно эти строки.

## Где менять первый экран и основные текстовые блоки

Откройте `index.html`:

- Первый экран находится в блоке `<header class="hero" id="top">`.
- Блок «С какой задачей приходят» находится в секции `<section class="routes" id="start">`.
- Блок «Обо мне» находится в секции `<section class="about">` ближе к низу страницы.
- Контакты находятся в секции `<section class="final-cta">`.

## Как изменить карточки направлений работы

Карточки в разделе «Направления работы» хранятся в `opportunities.js` внутри массива `opportunities`.

Формат одной карточки:

```js
{
  title: 'Название направления',
  category: 'упаковка',
  tags: ['тег 1', 'тег 2'],
  shortDescription: 'Короткое описание карточки.',
  forWhom: 'Для кого это направление.',
  whatICanDo: 'Что можно сделать в рамках задачи.',
  result: 'Какой результат получит клиент.',
  ctaText: 'Текст кнопки'
}
```

Важно: фильтры работают как точные категории. Если добавить новую категорию в карточку, нужно добавить такую же кнопку-фильтр в `index.html` с атрибутом `data-category="название категории"`.

## Как работают поиск и фильтры

- Кнопки-фильтры показывают карточки только выбранной точной категории.
- Поле поиска ищет шире: по названию, категории, тегам, описанию и текстам внутри карточек.
- Кнопка «сбросить» очищает поиск и выбранный фильтр.

Логика находится в `script.js`.

## Как заменить контакты

Сейчас в `index.html` и карточках `opportunities.js` указаны:

- Telegram: `@omnuka` со ссылкой `https://t.me/omnuka`
- Email: `omnuka@gmail.com` со ссылкой `mailto:omnuka@gmail.com`

Чтобы заменить Telegram или email, поменяйте ссылку и видимый текст в финальном блоке `index.html`. Если нужно изменить ссылки в кнопках карточек, отредактируйте `script.js` или шаблон карточки в функции `createCard`.

## Как изменить блок «Обо мне»

Блок находится в `index.html`:

```html
<section class="about" aria-labelledby="about-title">
```

Внутри можно менять заголовок и текст. Главный бренд сайта — Veromnuka, поэтому блок «Обо мне» расположен ближе к низу и не является главным экраном.

## Проверка перед публикацией

Запустите проверки JavaScript:

```bash
node --check script.js
node --check opportunities.js
```

После текстовых изменений также проверьте в браузере:

- поиск по карточкам;
- точные фильтры по категориям;
- кнопку «сбросить»;
- мобильную версию на узком экране.
