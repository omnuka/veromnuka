# Личный сайт Вероники Житниковой

Это простая статическая версия личного сайта-навигатора. Сайт помогает быстро понять, с какими задачами можно прийти к Веронике: дизайн упаковки, брендинг, ИИ-визуалы, контент, сайты, автоматизация и консультации.

## Как открыть сайт

1. Скачайте или откройте папку проекта на компьютере.
2. Найдите файл `index.html`.
3. Откройте его двойным кликом в браузере.

Ничего устанавливать не нужно: сайт работает без сервера, базы данных и платных библиотек.

## Из каких файлов состоит сайт

- `index.html` — структура страницы и основные блоки.
- `styles.css` — внешний вид сайта: цвета, сетка, карточки, мобильная версия.
- `script.js` — поиск, фильтры и вывод карточек на страницу.
- `opportunities.js` — список всех карточек каталога.
- `README.md` — эта инструкция.

## Как изменить текст на странице

Откройте файл `index.html` в любом текстовом редакторе и замените нужный текст. Например, там находятся первый экран, блок «Обо мне» и финальный призыв написать в Telegram.

## Как добавить новую карточку

1. Откройте файл `opportunities.js`.
2. Скопируйте любую карточку внутри массива `opportunities`.
3. Вставьте копию перед закрывающей строкой `];`.
4. Измените поля карточки:

```js
{
  title: 'Название направления',
  category: 'контент',
  tags: ['тег 1', 'тег 2'],
  shortDescription: 'Короткое описание карточки.',
  forWhom: 'Для кого эта возможность.',
  whatICanDo: 'Что можно сделать в рамках задачи.',
  result: 'Какой результат получит человек.',
  ctaText: 'Текст кнопки'
}
```

Важно: если карточка не последняя в списке, после закрывающей фигурной скобки должна стоять запятая.

## Как работают поиск и фильтры

- Поиск проверяет название, описание, категорию, теги и текст внутри карточки.
- Фильтр показывает карточки только выбранной категории.
- Кнопка «сбросить» очищает поиск и выбранный фильтр.

## Как заменить контакты

Сейчас в `index.html` и `script.js` указаны контакты-заглушки: Telegram `@omnuka` и email `omnuka@gmail.com`. Чтобы заменить Telegram, поменяйте ссылку `https://t.me/omnuka` и текст кнопки. Чтобы заменить email, поменяйте ссылку `mailto:omnuka@gmail.com` и текст в финальном блоке.

## Куда вставить изображения

На сайте уже есть безопасные места-заглушки для будущих изображений:

- Портрет на первом экране находится в `index.html` внутри блока `.portrait-card`. Чтобы заменить заглушку, вставьте внутрь `.portrait-placeholder` тег изображения, например:

```html
<img src="images/portrait.jpg" alt="Вероника Житникова">
```

- Блок «Избранные проекты» находится в `index.html` в секции `#projects`. В каждой карточке `.project-card` сейчас есть пустой декоративный `<div>` — его можно заменить на изображение кейса:

```html
<img src="images/project-01.jpg" alt="Упаковка продукта">
```

- Удобнее всего создать папку `images` рядом с `index.html` и положить туда файлы `portrait.jpg`, `project-01.jpg`, `project-02.jpg`, `project-03.jpg`, `project-04.jpg`.

После добавления реальных картинок можно заменить заголовки и описания карточек проектов на названия кейсов, задачу и результат.
